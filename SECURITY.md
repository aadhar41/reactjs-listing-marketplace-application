# Security Policy

## Supported Versions

Currently supported versions for security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please report it to us directly.

**Please do not report security vulnerabilities through public GitHub issues.**

Provide the following details:

- Description of the vulnerability.
- Steps to reproduce.
- Potential impact.

We will acknowledge your report within 48 hours. Thank you!
